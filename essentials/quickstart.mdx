# Quickstart

Copy the block below and run the commands from your project root.

```bash
# enter the project folder
cd your-project-folder

# install dependencies
npm install

# create local environment file
cp .env.example .env.local

# start dev server
npm run dev

# build for production
npm run build
```

“Your project may include specific environment variables; refer to your delivery documentation for those.”
