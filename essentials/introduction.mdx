# Introduction

This page explains the project's license system at a high level and where to place the license key in your environment.

---

## Where to put the license key

Add the license key to your `.env.local` file using the variable name provided in your delivery (commonly `LICENSE_KEY`). Restart the app after adding it.

Use the variables and paths provided in your delivery package.
