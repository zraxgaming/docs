<Section title="Start Fast" subtitle="Copy, paste, run">

```bash
# from project root
cp .env.example .env.local
npm install
npm run dev
```

<Tip icon="light-bulb">If your project uses `yarn` or `pnpm`, swap the install command accordingly.</Tip>

<Note icon="info">
Your project may include deployment-specific environment variables — use the variables and paths provided in your delivery package.
</Note>

</Section>
